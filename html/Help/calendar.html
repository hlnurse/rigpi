Calendar

RigPi Calendar is the calendar display for RigPi users. Set up events for Calendar in the Scheduler settings window.  Calendar events include reservations for rig use and custom events such as club meetings or skeds. You can overlay the Calendar with event lists from external sources, such as operating contest schedules.

 

 

Hold the mouse pointer over an event to see a tooltip with more details. The day with a yellow background is today's date.

 

 

The Calendar Week view shows all events for the selected week.

 

 

RigPi Calendar Day view

 

 

Drag the event line to change the event start time in the Event Scheduler.  Drag the bottom edge of an event to adjust the end time for that event in Event Scheduler.

 

 

 

Radios found on RigPi are shown on the login  screen, plus any upcoming reservations.
